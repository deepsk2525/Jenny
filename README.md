# Jenny — AI Virtual Voice Assistant

## Overview

Jenny is an AI-powered virtual voice assistant capable of handling voice commands, performing web searches, and executing intelligent automated tasks.

The project focuses on conversational AI workflows and intelligent task orchestration using Python and backend automation systems.

---

## Features

* Speech-to-text processing
* Text-to-speech responses
* Intelligent web search
* Automated task execution
* Flask-based backend APIs
* Modular architecture for future scalability

---

## Technologies Used

* Python
* Flask
* APIs
* Speech Recognition
* Automation Frameworks

---

## Installation

```bash
git clone https://github.com/deepsk2525/Jenny-AI-Assistant.git
cd Jenny-AI-Assistant
pip install -r requirements.txt
```

---

## Usage

```bash
python app.py
```
